## 그래픽 스타일 가이드

### 목표
- **지브리풍**: 따뜻하고 감성적인 2D 일러스트
- **색상**: 부드럽고 자연스러운 톤 (파스텔 계열 권장)
- **선 유지**: 명확하고 일관된 외곽선

### 도구
- **나노바나나2** — 배경 맵 생성 (AI 기반)
- **Aseprite** — 픽셀아트 캐릭터/오브젝트
- **Tiled Map Editor** — 타일맵 구성

## 플레이어 캐릭터

### 스펙
- **해상도**: 128x128px (타일 기준 2x2)
- **포즈**: 8방향 이동 애니메이션 (Up, Down, Left, Right + 대각선)
- **애니메이션 프레임**: 4프레임/동작
- **학년별 커스터마이징**: 머리색, 복장으로 학년 구분

### 학년별 색상
| 학년 | 머리색 | 복장 |
|------|--------|------|
| 1학년 | 주황색 | 흰 셔츠 |
| 2학년 | 갈색 | 노란 셔츠 |
| 3학년 | 핑크색 | 파란 셔츠 |
| 4학년 | 보라색 | 초록 셔츠 |
| 5학년 | 어두운파란 | 빨간 셔츠 |
| 6학년 | 검은색 | 검은 셔츠 |

## NPC

### 스펙
- **해상도**: 128x128px
- **역할**: 퀘스트 부여자, 상점 판매자, 스토리 NPC
- **스타일**: 캐릭터보다 독특한 옷/모자/액세서리로 구분

### 예시
- 수학 맵: "수학 마법사" NPC
- 국어 맵: "책장 지기" NPC
- 등등

## 몬스터/퀘스트 오브젝트

### 스펙
- **해상도**: 64x64px ~ 128x128px (다양한 크기)
- **개념**: 실제 몬스터보다 "교과 개념화" (예: 수 몬스터, 도형 몬스터)
- **애니메이션**: 유휴, 공격, 피격, 사망

### 예시 (수학)
- 🔢 **숫자 몬스터** — 각 숫자(1~9)를 의인화
- 📐 **도형 몬스터** — 삼각형, 사각형, 원 등의 도형
- ➕ **연산 몬스터** — 더하기, 빼기, 곱하기, 나누기 기호

## 아이템

### 스펙
- **해상도**: 64x64px
- **타입**: 장비, 소비, 재료, 학습 도구

### 예시

- 🧪 마법약 — 체력 회복
- 🔑 열쇠 — 퀘스트 진행 도구
- 💎 루비/에메랄드 — 화폐

## UI 요소

### 창 UI (Window Frame)
- **색상**: 고급스러운 갈색/파란색 테두리
- **글꼴**: 둥근 폰트 (초등학생 친화적)
- **버튼**: 눌렀을 때 피드백 (그림자, 스케일 변화)

### 요소 목록
- 📋 퀘스트 창
- 🎒 인벤토리
- 📍 미니맵
- 💬 대사 창
- ⭐ 경험치/레벨
- 🏥 체력 게이지

## 맵 제작

### 나노바나나2로 배경 생성
1. 콘셉트 입력 (예: "마법사의 탑, 따뜻한 분위기")
2. AI 생성 이미지 다운로드
3. Photoshop/GIMP로 경계선 정리

### Tiled Map Editor 타일맵 구성
1. 생성된 배경 이미지를 타일셋으로 변환
2. 충돌 영역 설정 (벽, 나무 등)
3. 오브젝트 배치 (NPC, 몬스터, 아이템)
4. 사운드 영역 표시 (BGM 구간)
