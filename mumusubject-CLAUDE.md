## 🌟 프로젝트 개요

- **타입**: 독립 교육용 JRPG 플랫폼

- **상태**: 개발 시작 예정

## 🎨 비전 & 컨셉

### 게임 스타일
- **그래픽**: 지브리풍 (따뜻하고 감성적인 2D 일러스트)
- **시점**: 탑뷰 (Top-down) — 성검전설 3 스타일
- **장르**: 교육용 JRPG + 협동 퀘스트
- **분위기**: 판타지 세계관 속에서 교과 지식으로 퀘스트를 해결

### 핵심 경험
- 학생이 캐릭터를 직접 조작하며 맵을 탐험
- 교과 학습 도구가 퀘스트/미션 형태로 자연스럽게 녹아있음
- 친구들과 함께 실시간으로 협동 퀘스트 해결
- 학습이 아닌 "놀이"처럼 느껴지는 경험 설계

---

## 🛠️ 기술 스택

### 게임 엔진
- **Phaser.js** — 브라우저 기반 2D 게임 엔진
  - 탑뷰 이동, 충돌 처리, 애니메이션
  - Tiled Map Editor와 호환

### 멀티플레이어
- **Socket.io + Node.js** — 실시간 서버
  - 동시접속 최대 20명
  - 실시간 위치 동기화
  - 협동 퀘스트 상태 공유

### 백엔드
- **Supabase** — DB, Auth, Storage
  - 학생 계정 관리
  - 퀘스트 진행도 저장
  - 학년/교과별 콘텐츠 데이터

### 배포
- **Vercel** — 프론트엔드
- **Railway** — Socket.io 서버

### 그래픽 제작
- **나노바나나2** — 배경 맵 이미지 생성 (지브리풍)
- **Tiled Map Editor** — 타일맵 구성
- 캐릭터, NPC, 오브젝트 → 직접 제작 (SKILLS.md 참조)

## 🗺️ 맵 구성 전략

### 배경 맵
- 나노바나나2로 지브리풍 배경 이미지 생성
- Tiled Map Editor로 타일맵 구성
- 교과별로 별도 맵 세계관 설계
  - 예) 수학 → 숫자 마법사의 탑, 국어 → 이야기 숲 등

### 오브젝트 (직접 제작)

- NPC (퀘스트 부여자)
- 아이템 (학습 보상)
- UI 요소 (퀘스트창, 인벤토리, 미니맵)
- → 상세 제작 가이드는 **SKILLS.md** 참조

## 🎯 수학 맵 첫 번째 개발 계획

### 벤치마킹
- **킹수학** (https://www.킹수학.com) — 전자칠판 학습도구 플랫폼
- 킹수학의 학습 도구를 JRPG 퀘스트로 재해석

### 수학 퀘스트 예시
- 연산 퀘스트 (사칙연산 몬스터 처치)
- 도형 퀘스트 (도형 미로 탈출)
- 측정 퀘스트 (왕국 건설 미션)
- 확률/통계 퀘스트 (보물 지도 해독)

### 학년별 난이도 분기
- 1~2학년: 기초 연산, 모양
- 3~4학년: 분수, 소수, 도형
- 5~6학년: 비율, 방정식, 통계

## 🌐 멀티플레이어 설계

### 서버 스펙
- 동시접속: 최대 20명 (학급 1반 기준)
- 룸 시스템: 학급별 독립 룸
- 실시간 동기화: 위치, 퀘스트 상태, 채팅

### 협동 퀘스트 구조
- 개인 퀘스트: 혼자 풀 수 있는 학습 문제
- 협동 퀘스트: 2~4명이 함께 역할 분담하여 해결
- 클래스 퀘스트: 학급 전체가 함께 해결하는 보스 퀘스트
