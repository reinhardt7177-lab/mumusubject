## 🎯 게임 방법

1. 링크 접속 → 자동으로 **3~4학년 모드**로 시작
2. 방향키(또는 화면 터치 조이스틱)로 캐릭터 이동
3. 몬스터에게 가까이 가면 **수학 문제 출제**
4. 정답을 입력하면 몬스터 처치 → 경험치·코인 획득
5. 모든 몬스터 처치 → 다음 방으로 이동
6. **보물방**에서 5문제를 모두 풀면 특별 보상!

## ✨ 주요 기능

- 🗺️ **4개의 방** — 점점 어려워지는 스테이지 구성
- 🔐 **비밀의 방** — 몬스터 전멸 후 북쪽 비밀 입구 개방
- 🏆 **레벨업 시스템** — 문제를 풀수록 캐릭터가 성장 (4단계)
- 💰 **코인 시스템** — 보물방 제단에서 5문제 도전으로 코인 획득
- 📱 **모바일 완벽 지원** — 태블릿/스마트폰 터치 조이스틱 + 숫자 키패드
- 🔤 **한국어 완전 지원**
